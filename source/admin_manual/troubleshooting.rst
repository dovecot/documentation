===============
Troubleshooting
===============

For debugging crashes, see https://dovecot.org/bugreport-mail

You can also use ``dovecot-sysreport`` tool to send information about the
system having troubles. Before posting it publicly, make sure the exported
configuration doesn't have anything sensitive in it.
