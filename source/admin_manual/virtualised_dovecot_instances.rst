===============================
Virtualised Dovecot instances
===============================

Dovecot can be run directly on physical hardware or virtualised using various virtualization technologies. 

Some customers are using OpenStack, some using KVM or VMware, we are in the end agnostic to the virtualization technology used, as long as there is no overcommitment of resources as explained more in detail in.
