.. _plugin-listescape:

=================
listescape plugin
=================

.. seealso:: See :ref:`listescape_plugin` for configuration information.

Settings
========

``listescape_char``
-------------------

- Default: ``\``
- Values:  :ref:`string`

The escape character to use.

``%`` needs to be written twice to escape it, because :ref:`config_variables`
are expanded in plugin section.
